### 6. 开发者公开 Codex 任务干预（Steer）与排队（Queue）实用技巧

*   **核心动态**：宝玉（@dotey）分享了在使用 Codex App 执行耗时任务时的两项高级干预机制：**Steer（干预）**和**Queue（排队）**。这大幅提升了开发者在使用智能体（Agent）开发时的流畅度和沟通效率。
*   **要点速览**：
    *   **Steer（干预）**：快捷键 `Shift + Enter`，在 Codex 任务未执行完毕时可中途打断，并指引新的执行方向。
    *   **Queue（排队）**：允许在当前任务仍在执行时，在下方堆叠并排队新的 Prompt 提示，在当前任务结束后自动按序执行，避免开发者盲目等待。

*   **要点速览**：
    *   **免去繁琐依赖**：完全屏蔽了 Python 复杂的底层环境安装，用户只要有 Go 和 Docker 即可直接使用。
