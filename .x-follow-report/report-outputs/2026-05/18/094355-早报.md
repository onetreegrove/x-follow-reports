### 🤖 藏师傅演示“工作流全家桶”：基于文档一键生成动效讲解视频
**内容摘要**：
归藏 (@op7418) 展示了一组极其硬核的 AI 工具组合拳：使用 **PPT Skill + Codex + HyperFrames**，实现了从纯文字文档到带有动效、配音及演示片段的完整产品介绍视频生成。

*   **技术细节**：该流程中 Codex 负责逻辑编排并在聊天框内直接预览视频，HyperFrames 负责渲染，配音由 Listenhub Skill 完成，极大地降低了开发者制作宣传视频的门槛。
*   **资源推荐**：Anthropic 官方发布了最新的 **Skill 构建指南**，藏师傅同步分享了 AI 翻译的双语版本，推荐所有关注 Agent 扩展的开发者研读。

### 🛠️ 微信读书 CLI 爆火，Agent 友好型 SaaS 成为新趋势
**内容摘要**：
@vista8 分享了微信读书 CLI (Weread Skill) 的详细配置方案。用户现在可以通过 Codex 或 Claude Code 直接调用该 Skill 来查询笔记、划线及阅读统计，甚至能基于个人阅读数据进行深度分析。

*   **实战提示**：官方与第三方 (@eviljer) 均提供了 Skill 版本，可通过 `npx skills add jerlinn/jerlin-weread` 快速安装。此动向说明传统办公/内容应用正在加速“Skill 化”，以接入 AI 代理生态。
*   **新工具集锦**：
    *   **netviz**: 浏览器端拖拽式网络架构图工具，支持免代码搭建服务器、数据库流向图。
    *   **Zoo Code**: Roo Code 停止开发后的延续者，由核心贡献者接手，在 VS Code 中提供完整的 AI 编程团队。
    *   **HermesPet**: 让 AI 以“桌宠”形式常驻 MacBook 刘海下方，支持多模态交互。
